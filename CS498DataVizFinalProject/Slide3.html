Regional Improvements (2010-2015)

Between 2010 and 2015, efforts to control the number of malaria cases and malaria deaths were successful in an all WHO regions. The number of cases in South East Asia in particular, saw a remarkable reduction of 54%. This also corresponded to a decrease of 46% in the Malaria related deaths in this region. The Western Pacific region showed the most improvement in controlling the number of malaria related deaths, with a reduction of 58%. The Eastern Mediterranean region showed the slowest improvment in curtailing both the number of Malaria cases, and the number of Malaria deaths. This region had a reduction of 11% in the number of cases, and 6% in the number of Deaths.

Use the User Input buttons below to view the Regional Percentage Reduction in Malaria Cases, or the Regional Percentage Reduction in Malaria Deaths, for the 2010-2015 time period.

About the Visualization

This Visualization follows the Interactive Slide show hybrid structure. Slide 3 depicts a Scene that conveys to the user, the Improvements that have been made in each of the WHO regions, with respect to reducing the number of cases of malaria, and the number of deaths caused by Malaria. The Visualization in this scene is a bar chart, where each region is represented by a uniquely colored bar. The colors of the bars in this chart, match the colors of the circles in the previous chart, providing the user with a consistent visual structure when viewing this Interactive Slide show as a whole. The size of each of the bars in the chart represents the percentage reductions that have been made in the number of Malaria Cases/Malaria Deaths between 2010 and 2015, depending on which chart the user has selected to view. The Triggers in this Visualization include the User's selection of a button, that determines whether they want to view a graph of the Percentage Reduction in Malaria Cases by Region, or a graph of the Percentage Reduction in Malaria Deaths by Region. This selection drives a Parameter that transitions between the two datasets, and animates the Visualization to display the graph that the user wishes to see. Additional Triggers in this scene include Mouse Over and Mouse Out events. When a user Mouses over a bar on the graph, it displays a Tool Tip Annotation , that shows the Region selected, and the percentage of Reduction in Malaria cases/deaths in that Region. It also changes the opacity of the bar that was moused over. The Graph is also Annotated with axes to show the Regions on the x-axis, and the percentage reductions on the y-axis. Additional Parameters in this Interactive Slide show are defined by the Slide Numbers that the User can click on, to advance the presentation. These are provided both in a navigation bar at the top of the page, and as Navigation buttons at the bottom of the page. These Parameters determine which slide is displayed to the user. The user can Trigger a change in the scenes contained within the Slide show, by modifying these Parameters.
